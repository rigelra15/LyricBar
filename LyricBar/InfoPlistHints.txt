Add the following to your target's Info.plist:

- NSAppleEventsUsageDescription (String): "LyricBar needs access to control Music/Spotify to read the current song for synchronized lyrics."

If your app is sandboxed and you fetch from the internet (LRCLIB), ensure:
- com.apple.security.network.client = YES (Outgoing Connections (Client))

Also, when first accessing Music/Spotify via AppleScript, macOS will prompt the user to allow automation. Ensure you test on a real macOS environment (not Playgrounds).
